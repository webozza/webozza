Accept bitcoin on your WooCommerce store

Accept bitcoin in just a few minutes with the ColtPay Commerce Payment Gateway for WooCommerce. Get up and running by following these simple steps:

  • Download and install the ColtPay Commerce for WooCommerce payment gateway
  • Link ColtPay Commerce with your WooCommerce store

Download and Install ColtPay Commerce for WooCommerce

To download and install the ColtPay Commerce Payment Gateway for WooCommerce,
navigate to Plugins > Add New within your WordPress admin page.
Search for "ColtPay Commerce". You should see the ColtPay Commerce Payment Gateway for WooCommerce as a top result. Click on the Install Now button to install the plugin. Once it has been installed, click Activate:

Link ColtPay Commerce with your WooCommerce Store

Within the WordPress administration area, go to the WooCommerce > Settings > Payments page and you will see ColtPay in the list of available payment gateways.

Click on the Manage button on the right hand side. This will take you to the settings page, where you can configure the plugin for your store. You should see something like this:

To link your accounts, you’ll need to add API Key from your coltpay account.

You will get your API Key from your ColtPay Merchant account. Go to Settings > Api Reference

That’s it! Your customers are now able to checkout and pay with Bitcoin
