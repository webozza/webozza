Basic Wordpress Integration

This guide will describe the simpliest wordpress Integration using a shortcode.

Example

Below is an example for a finished coltpay payment button.

1 Account Profile

Go to Account Profile > API keys and generate a new Public API key. These keys can not be viewd once they are created so we sure to save it in a safe location. The Public API key can only be used to create invoices and is safe to put on your website.

2 WP Admin Panel

Inside your WP admin panel go to Plugins > Add New. In the search bar look up "ColtPay Commerce" and click install. After the plugin is installed click Activate.

3 WP Settings

Once the plugin is installed and activated you can find the settings page under Settings > ColtPay.

4 WP Settings - API information

On this page you'll see two input boxes for the API keys found in Account Profile > API Keys. The public key is required. You do not need to enter the private key for the basic installation.
